# Contributing

Thanks for your interest in contributing! Here's how to get started.

## Development Setup

```bash
git clone https://github.com/frankliu20/<repo>.git
cd <repo>
npm install
npm run compile
```

Press `F5` in VS Code to launch the Extension Development Host.

## Workflow

1. **Fork** the repo and create a branch (`feat/my-feature` or `fix/my-bug`)
2. Make your changes — follow existing code style
3. Run `npm run compile` and `npm run lint` to verify
4. Commit using [Conventional Commits](https://www.conventionalcommits.org/) (`feat:`, `fix:`, `docs:`, etc.)
5. Open a **Pull Request** — fill out the template

## Guidelines

- Keep changes minimal and focused
- One logical change per PR
- Don't change dependency versions without discussion
- Add/update tests when possible

## Reporting Bugs

Use the [Bug Report](../../issues/new?template=bug_report.yml) template.

## Requesting Features

Use the [Feature Request](../../issues/new?template=feature_request.yml) template.
