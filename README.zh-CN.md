# CodeZen 🪵

> 写代码攒功德，把编码变成一场修行。

[English](./README.md)

## 功能特性

- **功德计数** — 每次击键积累功德，状态栏实时显示
- **木鱼动画** — 侧边栏木鱼敲击动画，金色波纹 + 浮动「功德 +1」
- **防刷机制** — 只计真实输入（粘贴、自动补全、宏脚本均被过滤）
- **七级境界** — 从施主修行至编程成佛，晋升时弹出祝贺通知
- **可选音效** — 木鱼敲击声，可配置触发概率（默认关闭）
- **中英双语** — 完整支持中文和英文界面
- **纯本地** — 零数据上传，所有数据通过 VS Code globalState 本地存储

## 境界系统

| 等级 | 境界 | 所需功德 |
|------|------|---------|
| 1 | 施主 | 0 |
| 2 | 沙门 | 1,000 |
| 3 | 码僧 | 10,000 |
| 4 | 禅师 | 50,000 |
| 5 | 代码罗汉 | 200,000 |
| 6 | 架构菩萨 | 1,000,000 |
| 7 | 编程成佛 | 10,000,000 |

## 快速上手

1. 安装扩展
2. 状态栏出现 `🪵 0` — 开始写代码即可积攒功德
3. 点击左侧活动栏的 CodeZen 图标，打开木鱼动画面板
4. 点击状态栏功德数字，查看功德簿

## 命令

| 命令 | 说明 |
|------|------|
| `CodeZen: Show Merit Book` | 查看累计功德、当前境界、距下一境界所需功德 |
| `CodeZen: Toggle Sound` | 开关木鱼音效 |
| `CodeZen: Reset Merit` | 清零所有功德（需二次确认） |

## 设置项

| 设置 | 默认值 | 说明 |
|------|--------|------|
| `codezen.sound.enabled` | `false` | 是否启用木鱼音效 |
| `codezen.sound.probability` | `0.01` | 每次击键触发音效的概率（0–1） |
| `codezen.statusBar.format` | `🪵 {merit}` | 状态栏格式，可用变量：`{merit}`、`{rank}` |
| `codezen.locale` | `auto` | 显示语言：`auto`（跟随 VS Code）、`en`、`zh-CN` |

## 防刷规则

- 仅单字符输入计数（粘贴和批量插入不计）
- 同一按键连续按超过 5 次后概率衰减
- 每分钟上限 300 功德（防按键精灵）
- 退格 / 删除不扣分（慈悲为怀 🙏）
